# React Native Template

This is a React Native template with Expo and TypeScript.

## Features

- [x] Expo
- [x] TypeScript
- [x] ESLint
- [x] Prettier
- [x] Husky
- [x] Lint-staged
- [x] React Compiler
- [x] ESLint React Compiler
- [x] i18n-js
- [x] Toast
- [x] Unistyles
- [ ] Notifications (Expo Push Notifications)
- [ ] Zustand (State Management)
- [ ] Supabase (Auth and Database) or Proprietary backend API.
